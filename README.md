# FinSecure-ML-Pipeline: Imbalanced Data Classification

## Overview
An end-to-end machine learning pipeline designed to handle highly skewed tabular datasets. This project covers the entire ML lifecycle: from automated preprocessing and synthetic data generation to model training, tracking, and API deployment.

## Features (Planned)
- **Automated Preprocessing:** Pipeline scripts for handling missing values and implementing techniques like SMOTE or custom class weighting.
- **Model Training:** Gradient boosting implementation utilizing CatBoost/XGBoost for optimal performance on tabular data.
- **Experiment Tracking:** MLflow integration to log model parameters, metrics (Precision-Recall AUC), and artifacts.
- **Serving:** Containerized FastAPI endpoint for low-latency inference.

## Tech Stack
- **Modeling:** Scikit-Learn, CatBoost
- **MLOps:** MLflow, Docker
- **Deployment:** FastAPI, Uvicorn
